# Azure DevOps Queries

## Lesson 35

### ADO Queries
A versatile tool that allows you to search and filter work items within Azure DevOps.  
It helps teams improve their workflow management and gain better visibility into their projects.

### Query Types
+ Flat
+ Tree
+ Linked

### Benefits of Using Queries
+ Prioritization and Planning
+ Cross-Project Queries
+ Query Tags
+ Using Query Reports
+ Tracking Dependencies
+ Quality Assurance
+ Building Dashboards

---

## Lesson 36

### Creating a Flat Query

+ Go to **Boards → Queries**.
+ You can create a new query or edit an existing one.
+ Select **Fields**, **Operator**, **And/Or clauses**, and **Values**.
+ The results displayed are **Work Items**.

---

## Lesson 37

### Linked Queries
Displays all associated links between work items.

---

## Lesson 38

### Tree Query
Shows parent-child (father-son) relationships between work items.

---

## Lesson 39

### Charts
You can create and edit charts directly from query results.

### Reports
You can customize the query results table using **Column Options**.
